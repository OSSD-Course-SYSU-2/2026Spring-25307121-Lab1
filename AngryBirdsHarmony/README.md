# 愤怒的小鸟 - 鸿蒙版

这是一个基于鸿蒙系统开发的愤怒的小鸟游戏。

## 项目结构

```
AngryBirdsHarmony/
├── entry/                          # 主模块
│   └── src/
│       └── main/
│           ├── ets/                # ArkTS代码
│           │   ├── pages/          # 页面
│           │   │   └── Index.ets   # 主游戏页面
│           │   └── common/         # 公共模块
│           │       ├── Bird.ets    # 小鸟类
│           │       ├── Pig.ets     # 猪类
│           │       ├── Obstacle.ets # 障碍物类
│           │       └── GameEngine.ets # 游戏引擎
│           ├── resources/          # 资源文件
│           │   └── base/
│           │       ├── element/    # 字符串和颜色资源
│           │       ├── media/      # 图片资源
│           │       └── profile/    # 配置文件
│           └── module.json5        # 模块配置
├── build-profile.json5             # 构建配置
├── hvigorfile.js                   # Hvigor构建脚本
└── oh-package.json5                # 项目配置
```

## 游戏特性

### 核心玩法
- 🎯 **弹弓发射**：触摸拖拽小鸟，松开发射
- 📍 **轨迹预测**：拖拽时显示预测轨迹
- 🎮 **物理引擎**：重力、摩擦、碰撞检测

### 游戏元素
- 🐦 **愤怒的小鸟**：红色小鸟，带愤怒表情
- 🐷 **绿色小猪**：游戏目标
- 🪵 **木块障碍物**：棕色，可被摧毁
- 🪨 **石块障碍物**：灰色，更坚固

### 游戏系统
- ⭐ **得分系统**：消灭猪 +1000分，木块 +100分，石块 +200分
- 🎯 **关卡系统**：多个关卡，难度递增
- 🔄 **生命系统**：每关3只小鸟

## 技术实现

### 开发语言
- **ArkTS**：鸿蒙系统的开发语言，基于TypeScript扩展

### 核心技术
- **Canvas 2D**：用于游戏图形渲染
- **物理引擎**：自定义实现重力、碰撞检测
- **触摸事件**：支持触摸屏操作

### 架构设计
- **GameEngine**：游戏主引擎，负责游戏循环和状态管理
- **Bird**：小鸟类，处理小鸟的绘制和物理运动
- **Pig**：猪类，处理目标对象的绘制和碰撞
- **Obstacle**：障碍物类，处理障碍物的绘制和破坏

## 如何运行

### 环境要求
- DevEco Studio 3.0 或更高版本
- HarmonyOS SDK 9 或更高版本

### 运行步骤
1. 使用 DevEco Studio 打开项目
2. 连接鸿蒙设备或启动模拟器
3. 点击运行按钮安装并启动应用

## 游戏操作

1. 触摸屏幕上的小鸟并向后拖拽
2. 松开手指发射小鸟
3. 消灭所有绿色小猪即可过关
4. 点击"重新开始"按钮可以重玩
5. 点击"下一关"按钮可以跳过关卡

## 开发说明

### 扩展关卡
在 `GameEngine.ets` 的 `loadLevel` 方法中添加新的关卡配置。

### 自定义游戏元素
- 修改 `Bird.ets` 自定义小鸟外观
- 修改 `Pig.ets` 自定义猪的外观
- 修改 `Obstacle.ets` 添加新的障碍物类型

### 调整物理参数
在 `GameEngine.ets` 中修改以下常量：
- `GRAVITY`：重力加速度
- `FRICTION`：摩擦系数
