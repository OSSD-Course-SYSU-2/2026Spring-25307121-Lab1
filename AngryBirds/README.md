




# 愤怒的小鸟 - 鸿蒙版

一个基于鸿蒙(HarmonyOS)开发的愤怒的小鸟游戏。

## 游戏特性

- 🐦 经典的愤怒的小鸟玩法
- 🎯 弹弓发射机制，支持拖拽瞄准
- 🧱 多种障碍物类型（木头、石头、玻璃）
- 🐷 可爱的猪猪目标
- 🎮 3个精心设计的关卡
- 💯 完整的得分系统
- 🎨 精美的游戏画面

## 游戏玩法

1. **瞄准**: 用手指拖动小鸟向后拉动弹弓
2. **发射**: 松开手指，小鸟将被发射出去
3. **目标**: 消灭所有的猪猪即可过关
4. **障碍物**: 
   - 🪵 木头: 中等强度，容易被击碎
   - 🪨 石头: 高强度，需要多次撞击
   - 🪟 玻璃: 低强度，一击即碎

## 得分规则

- 击中障碍物: +10分
- 消灭猪猪: +500分
- 剩余小鸟: 每只+1000分（过关时）

## 运行项目

### 前置要求

1. 安装 [DevEco Studio](https://developer.harmonyos.com/cn/develop/deveco-studio)
2. 配置鸿蒙开发环境
3. 安装 HarmonyOS SDK

### 运行步骤

1. 使用 DevEco Studio 打开项目目录 `AngryBirds`
2. 等待项目初始化和依赖安装
3. 连接鸿蒙设备或启动模拟器
4. 点击运行按钮启动应用

## 技术实现

### 核心模块

- **GameEngine**: 游戏引擎，负责物理模拟、碰撞检测、游戏逻辑
- **GameCanvas**: 游戏画布，负责渲染所有游戏对象
- **GameModels**: 游戏模型，定义小鸟、障碍物、猪等游戏对象

### 物理引擎

- 重力模拟
- 碰撞检测
- 速度和加速度计算
- 简单的反弹效果

### 渲染系统

- Canvas 2D 绘图
- 60fps 游戏循环
- 轨迹预测显示
- 动画效果

## 关卡设计

### 第一关
- 3只小鸟
- 简单的木质结构
- 1只猪

### 第二关
- 4只小鸟
- 混合材质结构（木头+石头+玻璃）
- 2只猪

### 第三关
- 5只小鸟
- 复杂的多层结构
- 3只猪
